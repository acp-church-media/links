# ACP Media Links

Short link redirects for ACP Church Media Department.

## Live URLs
- **Index:** `https://acp-church-media.github.io/links/`
- **Announcements:** `https://acp-church-media.github.io/links/announcements`

## Setup (One Time)

1. Go to [github.com](https://github.com) → sign in as `acp-church-media`
2. Click **"New repository"**
3. Name it exactly: `links`
4. Set to **Public**
5. Click **Create repository**
6. Upload all files from this folder into the repo
7. Go to **Settings → Pages → Source → Deploy from branch → main / root**
8. Click **Save** — your site will be live in ~1 minute

## Adding a New Short Link

1. Create a new folder with your short name (e.g. `bulletin`)
2. Copy any existing `index.html` from another folder
3. Replace the destination URL in **three places** inside that file
4. Add a new `<a class="link-card">` entry to the main `index.html`
5. Commit and push — live in seconds
